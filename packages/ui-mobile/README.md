# `@lss/ui-mobile`

Hybrid mobile UI adapter for **Capacitor/Cordova**.

Because hybrid apps render in a WebView, this package re-exports everything from `@lss/ui-web` and is intended for **mobile-only wrappers** (safe areas, mobile navigation patterns, etc).

## Usage

```tsx
import { Button } from '@lss/ui-mobile'
```

Global CSS:

```css
@import '@lss/ui-mobile/styles/tailwind.css';
```
